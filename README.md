## âœ¨ Features

### 1. Job Board
- Browse 12+ job listings from top companies (Google, Netflix, Stripe, etc.)
- Filter by experience level, work type, and location
- View detailed job descriptions and requirements

### 2. AI-Powered Application Form
- Apply to jobs with intelligent AI assistance
- Get AI-generated answers tailored to your profile
- **Thumbs up/down feedback** to improve AI suggestions
- Edit and customize AI suggestions before submitting

### 3. Application Dashboard
- Track all your job applications in one place
- Visual pipeline showing application progress
- Update application status (Submitted, Interview Requested, Offer Received, etc.)

### 4. Resume Analyzer
- Upload your resume for AI analysis (simulated)

- Complete onboarding with career preferences
- Manage personal information, skills, work experience
- Preferences auto-populated from onboarding
- Secure local storage
